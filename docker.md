# Инструкция по запуску с помощью Docker

1. Убедитесь, что у вас установлен Docker и Docker Compose.
2. Откройте терминал в корне проекта.
3. Выполните команду для сборки и запуска контейнеров:

```
docker-compose up --build
```

4. После запуска приложение будет доступно по адресу, указанному в настройках (обычно http://localhost:8000).
5. Для остановки контейнеров используйте:

```
docker-compose down
```

6. Для применения изменений в коде перезапустите контейнеры:

```
docker-compose up --build
```
